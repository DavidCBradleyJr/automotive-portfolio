   - Run: `npm create vite@latest temp-scaffold -- --template vanilla`
   - Move the scaffolded files (package.json, vite.config.js, src/, public/, .gitignore if not exists) into the project root
   - Remove the temp-scaffold directory
   - Edit package.json: set name to "automotive-portfolio", pin vite to "^6.0.0"
   - Run `npm install` to install Vite 6.x
   - Delete the default Vite scaffold files: src/counter.js, src/javascript.svg, public/vite.svg, and the default content in src/style.css and src/main.js

2. Create `src/tokens.css` with the three-tier CSS custom property system from RESEARCH.md:
   - Tier 1 (Primitives): All raw color values (#0F0F0F, #7C3AED, rgba variants, #E0E0E0, #9CA3AF, #FFFFFF), font families (Orbitron, Space Grotesk), spacing scale (xs through 2xl: 0.25rem to 8rem), typography scale (sm through hero using clamp for hero: clamp(3rem, 8vw, 6.25rem))
   - Tier 2 (Semantic): --color-bg, --color-text, --color-text-muted, --color-accent, --color-accent-glow, --color-border mapping to primitives
   - Tier 3 (Component): --nav-bg (rgba(15,15,15,0.85)), --nav-blur (12px), --nav-height (4rem), --hero-overlay gradient, --btn-primary-bg, --btn-primary-text
   - IMPORTANT: Purple (#7C3AED) contrast on #0F0F0F is only 3.36:1 -- FAILS WCAG AA for normal text. Only use as large text color, filled button backgrounds, borders, or glows. Body text must use #E0E0E0 (14.52:1 ratio).

3. Set up `src/main.js` to import `./style.css` only (no component imports yet -- Plans 02 and 03 will add those).

4. Create a minimal `vite.config.js` if needed (Vite vanilla template may not require one, but create if the scaffold generates one).

<task type="auto">
  <name>Task 2: Create global styles and complete HTML skeleton</name>
  <files>src/style.css, index.html, public/images/hero-placeholder.webp</files>

1. Create `src/style.css` with:
   - `@import './tokens.css';` at the top
   - Modern CSS reset (Josh Comeau style from RESEARCH.md): box-sizing border-box, margin 0, scroll-behavior smooth, line-height 1.6, font-smoothing, img/video block + max-width 100%, input/button font inherit, overflow-wrap break-word
   - `prefers-reduced-motion` media query: reduce all animation-duration and transition-duration to 0.01ms, scroll-behavior to auto
   - Base body styles: background var(--color-bg), color var(--color-text), font-family var(--font-body), font-size var(--text-base)
   - Typography: h1-h3 use var(--font-display) (Orbitron), h2 in text-transform uppercase with letter-spacing 0.05em
   - Button base class `.btn` with padding, border-radius, font-weight, cursor pointer, transition. `.btn--primary` with background var(--btn-primary-bg), color var(--btn-primary-text), hover glow effect using box-shadow with var(--color-accent-glow)
   - Section base class `.section` with min-height 50vh (for scroll-spy to work even with stub content), padding var(--space-xl) horizontal, max-width for content
   - Section heading `.section__heading` in Orbitron, uppercase, centered, with bottom margin
   - Responsive breakpoints: @media min-width 768px, 1024px, 1440px with appropriate adjustments
   - `.no-scroll` utility class: overflow hidden on body (for mobile menu)

2. Create `index.html` with:

   - Head: charset utf-8, viewport meta (width=device-width, initial-scale=1.0), title "David Bradley | Automotive Photography"

   - Open Graph placeholder meta tags (title, description, type=website)

   - Body contains:
     a. Navigation header (`<header class="nav" id="nav">`):
        - Inner container `.nav__container` with flex layout
        - Logo area `.nav__logo`: `<a>` with "DB" span `.nav__monogram` + "David Bradley" span `.nav__wordmark`
        - Desktop nav links `.nav__links`: `<ul>` with `<li><a>` for Gallery (#gallery), Video (#video), About (#about), BTS (#bts), Contact (#contact) -- each with class `.nav__link`
        - "Book a Shoot" CTA button `.nav__cta .btn .btn--primary` linking to #contact
        - Hamburger button `.nav__hamburger` with aria-label="Menu", aria-expanded="false", containing 3 span bars
        - Mobile overlay `.nav__overlay` with same links + CTA

     b. Hero section (`<section class="hero" id="hero">`):
        - `<img class="hero__image" src="/images/hero-placeholder.webp" alt="High-performance sports car photographed at golden hour" fetchpriority="high" width="1920" height="1080">` -- NO loading="lazy", NO loading attribute at all
        - `<div class="hero__overlay"></div>` for gradient overlay
        - `<div class="hero__content">` positioned bottom-left containing:
          - `<h1 class="hero__name">David Bradley</h1>`
          - `<p class="hero__tagline">Where Speed Meets Art</p>`
          - `<a href="#gallery" class="hero__cta btn btn--primary">View Gallery</a>`
