<task type="auto">
  <name>Task 1: Implement iOS-safe scroll lock with position save/restore</name>
  <files>src/style.css, src/components/nav.js</files>
  <action>
    1. In src/style.css, replace the `.no-scroll` utility (currently just `overflow: hidden`) with an iOS-safe scroll lock:

       ```css
       .no-scroll {
         overflow: hidden;
         position: fixed;
         width: 100%;
         /* top is set dynamically by JS to preserve scroll position */
       }
       ```

       The `position: fixed` prevents iOS Safari from allowing background scroll. The `top` value is set by JS to offset the body so the viewport does not jump.

    2. In src/components/nav.js, add a `savedScrollY` variable at the top of initNav() (after the element queries). Then update openOverlay() and closeOverlay():

       openOverlay():
       ```javascript
       function openOverlay() {
         savedScrollY = window.scrollY;
         document.body.style.top = `-${savedScrollY}px`;
         document.body.classList.add('no-scroll');
         hamburger.classList.add('nav__hamburger--open');
         overlay.classList.add('nav__overlay--open');
         hamburger.setAttribute('aria-expanded', 'true');
         const firstLink = overlay.querySelector('a');
         if (firstLink) {
           requestAnimationFrame(() => firstLink.focus());
         }
       }
       ```

       closeOverlay():
       ```javascript
       function closeOverlay() {
         hamburger.classList.remove('nav__hamburger--open');
         overlay.classList.remove('nav__overlay--open');
         document.body.classList.remove('no-scroll');
         document.body.style.top = '';
         window.scrollTo(0, savedScrollY);
         hamburger.setAttribute('aria-expanded', 'false');
         hamburger.focus();
       }
       ```

       Key points:
       - Save scrollY BEFORE adding no-scroll (position:fixed resets scroll to 0)
       - Set body top to negative scrollY so viewport content stays visually in place
       - On close, remove no-scroll FIRST, clear inline top, THEN scrollTo the saved position
       - Do NOT change any other behavior (hamburger click, overlay link click, Escape key, focus trapping)
  </action>
