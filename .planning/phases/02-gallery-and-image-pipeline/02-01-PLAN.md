<task type="auto">
  <name>Task 1: Create image processing build script</name>
  <files>scripts/process-images.js, package.json</files>
  <action>
Install sharp as a dev dependency:
```
npm install -D sharp
```

Create `scripts/process-images.js` -- a Node.js ESM script that:

1. Reads all JPG files from `~/Pictures/cars/` (the 29 DSC-numbered files).
2. For each real image:
   - Resizes to max 2000px width (withoutEnlargement: true) using sharp.
   - Converts to WebP at quality 82.
   - Outputs to `public/images/gallery/{descriptive-name}.webp`.
   - Generates a 16px LQIP WebP at quality 20, converts to base64 data URI.
   - Reads metadata (width, height) to store dimensions.
3. Distributes the 29 real images across 5 categories (roughly 5-6 per category):
   - jdm: 6 images
   - euro: 6 images
   - supercar: 6 images
   - american-muscle: 6 images
   - track: 5 images
4. Assigns descriptive filenames based on category and index (e.g., "jdm-drift-car-01", "euro-sports-coupe-02", "supercar-exotic-red-03"). Since we cannot see the actual photo content, use generic but descriptive automotive names per category.
5. Creates `public/images/gallery/` directory if it doesn't exist (use `mkdir -p` equivalent via `fs.mkdir({ recursive: true })`).
6. After processing real images, generates gradient placeholder entries for the remaining slots to reach 8-12 per category (gradient placeholders do NOT create files -- they use CSS gradients, so only data entries are needed).
7. Writes the complete gallery data to `src/data/gallery-images.js` as an ES module export.

The output data file format:
```javascript
export const galleryImages = [
  {
    id: 'jdm-street-racer-01',
    src: '/images/gallery/jdm-street-racer-01.webp',
    lqip: 'data:image/webp;base64,...',
    width: 2000,
    height: 1333,
    category: 'jdm',
    caption: 'JDM Street Racer',
    alt: 'Japanese domestic market street racing car',
    isPlaceholder: false,
  },
  {
    id: 'jdm-placeholder-07',
    src: null,
    lqip: null,
    width: 1600,
    height: 1067,
    category: 'jdm',
    caption: 'JDM Import',
    alt: 'Automotive photography placeholder',
    isPlaceholder: true,
    gradient: 'linear-gradient(135deg, #1a1a2e, #e94560)',
  },
  // ... 40-60 total entries
];

export const categories = [
  { id: 'all', label: 'All' },
  { id: 'jdm', label: 'JDM' },
  { id: 'euro', label: 'Euro' },
  { id: 'supercar', label: 'Supercar' },
  { id: 'american-muscle', label: 'American Muscle' },
  { id: 'track', label: 'Track/Motorsport' },
];
```
