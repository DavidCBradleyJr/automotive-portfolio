**initEditor() implementation:**

1. **Write/Preview tab switching:**
   - Query `.blog-editor-tab` buttons inside the editor
   - On click, toggle `blog-editor-tab--active` class
   - If "write" tab: show `#post-body` textarea, show `#blog-toolbar`, hide `#post-preview`
   - If "preview" tab: hide `#post-body` textarea, hide `#blog-toolbar`, show `#post-preview`, render markdown to HTML
   - For preview rendering: `import { marked } from 'marked';` then set the preview element content using marked.parse(textarea.value)
   - IMPORTANT: Since this is an admin-only panel behind auth (not public-facing), and the markdown is authored by David himself, the preview rendering of David's own markdown is acceptable. However, if you want extra safety, you can sanitize the marked output or use textContent for non-HTML parts.

2. **Formatting toolbar:**
   - Query all `#blog-toolbar .blog-toolbar__btn` buttons
   - Add click handler on each that reads `data-action` attribute
   - Implement `insertAtCursor(textarea, before, after)` using `selectionStart`/`selectionEnd`:
     ```javascript
     function insertAtCursor(textarea, before, after = '') {
       const start = textarea.selectionStart;
       const end = textarea.selectionEnd;
       const selected = textarea.value.substring(start, end);
       const replacement = before + (selected || 'text') + after;
       textarea.setRangeText(replacement, start, end, 'select');
       textarea.focus();
       textarea.selectionStart = start + before.length;
       textarea.selectionEnd = start + before.length + (selected || 'text').length;
     }
     ```
   - Toolbar action map:
     - bold: `insertAtCursor(ta, '**', '**')`
     - italic: `insertAtCursor(ta, '_', '_')`
     - h2: `insertAtCursor(ta, '## ', '\n')`
     - h3: `insertAtCursor(ta, '### ', '\n')`
     - link: `insertAtCursor(ta, '[', '](url)')`
     - blockquote: `insertAtCursor(ta, '> ', '\n')`
     - ul: `insertAtCursor(ta, '- ', '\n')`
     - ol: `insertAtCursor(ta, '1. ', '\n')`

3. **Slug auto-generation:**
   - On `#post-title` input event, if slug field has NOT been manually edited (track with a `slugManuallyEdited` flag)
   - Generate slug: `title.toLowerCase().replace(/[^a-z0-9]+/g, '-').replace(/^-|-$/g, '').slice(0, 60)`
   - Set `#post-slug` value to the generated slug
   - On `#post-slug` input event, set `slugManuallyEdited = true`

Replace the placeholder `src/admin/components/blog-manager.js` with the full implementation:

**Imports:**
```javascript
import { getToken } from './auth.js';
import { showToast } from './toast.js';
import { initEditor, getEditorData, setEditorData, clearEditor, buildMarkdown } from './blog-editor.js';
```

**Module state:**
```javascript
let posts = [];           // Array of post objects from list-posts
let currentPost = null;   // Post being edited (null for new post)
let saving = false;       // Prevent concurrent saves
```

**initBlogManager() -- main export:**
1. Call `initEditor()` to set up toolbar and preview
2. Call `loadPosts()` to fetch and render the post grid
3. Bind event listeners:
   - `#new-post-btn` click: call `openEditor(null)` (new post)
   - `#back-to-posts` click: call `closeEditor()`
   - `#save-draft-btn` click: call `savePost('draft')`
   - `#publish-btn` click: call `savePost('published')`

**loadPosts() -- fetch and render post list:**
- `const token = await getToken();`
- `fetch('/.netlify/functions/list-posts', { headers: { Authorization: 'Bearer ' + token } })`
- Parse response JSON, store in `posts` array
- Call `renderPostGrid()`

**renderPostGrid():**
- Get `#blog-post-grid` element
- If `posts.length === 0`: show "No posts yet. Click 'New Post' to create one."
- Otherwise, for each post create a card using DOM methods (createElement, textContent for text, setAttribute for attributes -- avoid setting raw HTML from untrusted sources):
  - Card container: `div.blog-post-card` with `data-slug` attribute
  - Cover image: `img.blog-post-card__cover` with src set to `post.cover` Cloudinary URL (append `?w=400&f_auto&q_auto` for thumbnail), or a placeholder div if no cover
  - Body container: `div.blog-post-card__body`
  - Title: `div.blog-post-card__title` with textContent = post.title
  - Meta row: `div.blog-post-card__meta` with date span (textContent) and status span (textContent, class `blog-post-card__status--${post.status}`)
  - Actions: `div.blog-post-card__actions` with a Delete button (`btn btn--secondary btn--small blog-post-card__delete`)
- Add click handler on each card (excluding delete button): call `openEditor(post)`
- Add click handler on delete buttons: call `deletePost(post)` with confirm dialog

**openEditor(post):**
- Set `currentPost = post`
- If post is null (new post): call `clearEditor()`, set date to today (YYYY-MM-DD format)
- If post exists: call `setEditorData(post)`, make slug field readonly (prevent orphaned files per pitfall 6)
- Hide `#blog-list-view`, show `#blog-editor-view` (toggle `hidden` attribute)
- Update button labels: if existing published post, change Publish button to "Update"

**closeEditor():**
- Set `currentPost = null`
- Show `#blog-list-view`, hide `#blog-editor-view`
- Make slug field editable again (remove readonly)
- Reload posts: call `loadPosts()`

**savePost(status):**
- If `saving` is true, return (prevent concurrent operations per pitfall 4)
- Set `saving = true`, disable both save buttons, show "Saving..." text
- Get editor data: `const data = getEditorData()`
- Validate: title required, slug required. If missing, `showToast('error', 'Title and slug are required')` and return
- Set `data.status = status`
- Set `data.date = data.date || new Date().toISOString().split('T')[0]`
- Build markdown: `const content = buildMarkdown(data, data.body)`
- POST to `/.netlify/functions/save-post` with body: `{ slug: data.slug, content, sha: currentPost?.sha, title: data.title }`
- On success: `showToast('success', status === 'published' ? 'Post published!' : 'Draft saved!')`, update `currentPost.sha` with returned sha, call `closeEditor()`
- On error: `showToast('error', 'Failed to save post: ' + error message)`
- Finally: set `saving = false`, re-enable buttons, restore button text

**deletePost(post):**
- Show confirm dialog: `confirm('Delete "' + post.title + '"? This cannot be undone.')`
- If cancelled, return
- Set `saving = true`
- POST to `/.netlify/functions/delete-post` with body: `{ slug: post.slug, sha: post.sha, title: post.title }`
- On success: `showToast('success', 'Post deleted')`, call `loadPosts()`
- On error: `showToast('error', 'Failed to delete post: ' + error message)`
- Finally: set `saving = false`
