Create `src/admin/components/blog-gallery-sidebar.js`:

**Export:** `initGallerySidebar()`

**initGallerySidebar() implementation:**

1. **Load gallery images:**
   - Fetch `/.netlify/functions/list-images` with JWT auth header (use `getToken()` from auth.js)
   - Parse response to get images array
   - Store images in module-level variable for reuse by cover picker
   - Call `renderSidebarImages(images)` to populate `#sidebar-gallery-grid`

2. **renderSidebarImages(images):**
   - Clear `#sidebar-gallery-grid`
   - For each image, create an `<img>` element using DOM methods:
     - `img.src` = Cloudinary thumbnail URL: take `image.secure_url` and add `/w_200,h_200,c_fill,f_auto,q_auto/` transform (or use the URL pattern from the existing gallery view)
     - `img.alt` = `image.alt || image.caption || ''`
     - `img.draggable = true`
     - `img.className = 'blog-gallery-sidebar__img'`
     - Store full-size URL on the element: `img.dataset.fullUrl` = image.secure_url (with `/w_1600,f_auto,q_auto/` transform for blog-quality images)
     - Store alt text: `img.dataset.alt` = image.alt || image.caption || ''
   - Add dragstart handler on each image:
     ```javascript
     img.addEventListener('dragstart', (e) => {
       const fullUrl = img.dataset.fullUrl;
       const alt = img.dataset.alt;
       e.dataTransfer.setData('text/plain', '![' + alt + '](' + fullUrl + ')');
       e.dataTransfer.effectAllowed = 'copy';
     });
     ```

3. **Textarea drop handler:**
   - Get `#post-body` textarea
   - Add `dragover` handler: `e.preventDefault(); e.dataTransfer.dropEffect = 'copy';`
   - Add `drop` handler:
     ```javascript
     textarea.addEventListener('drop', (e) => {
       e.preventDefault();
       const markdownImg = e.dataTransfer.getData('text/plain');
       if (!markdownImg.startsWith('![')) return; // Only handle our image drops
       const pos = textarea.selectionStart;
       const before = textarea.value.substring(0, pos);
       const after = textarea.value.substring(pos);
       const insertion = (before.endsWith('\n') || before === '' ? '' : '\n') + markdownImg + '\n';
       textarea.value = before + insertion + after;
       textarea.selectionStart = textarea.selectionEnd = pos + insertion.length;
       textarea.focus();
     });
     ```

4. **Sidebar toggle (collapse/expand):**
   - Bind `#sidebar-toggle` click to toggle the sidebar visibility
   - When hidden: add a CSS class `blog-gallery-sidebar--collapsed` (set `display: none` on sidebar grid and upload button), change button text to "Show"
   - When shown: remove class, change button text to "Hide"

5. **Upload button -- new photo upload in editor:**
   - Create a hidden `<input type="file" accept="image/jpeg,image/png,image/webp">` element
   - On `#sidebar-upload-btn` click, trigger the file input click
   - On file input change:
     a. Show a category prompt: use `prompt('Category for this photo (jdm, euro, supercar, american-muscle, track):')` -- simple and effective for a single-user admin
     b. If no category provided, default to 'blog' or show toast error and return
     c. Generate a public_id from filename (lowercase, replace non-alphanumeric with hyphens)
     d. Get signed upload params: POST to `/.netlify/functions/sign-upload` with `{ folder: 'portfolio/' + category, public_id }` and JWT auth
     e. Upload to Cloudinary: POST to `https://api.cloudinary.com/v1_1/${cloudName}/image/upload` with FormData (file, api_key, timestamp, signature, folder, public_id)
     f. The cloud name is `dl0atmtb7` (from existing Cloudinary URLs in the project)
     g. On success: show toast "Photo uploaded!", add the new image to the sidebar grid, insert the markdown image tag at cursor in textarea
     h. On error: show toast with error message
   - Read the sign-upload.mjs function to understand the exact request/response format for signed params

6. **Cover image picker:**
   - On `#pick-cover-btn` click: create a simple overlay/modal with all gallery images as a grid
   - Each image in the picker grid gets a click handler that:
     - Sets `#cover-preview` src to the clicked image's full-size Cloudinary URL (with `/w_1600,f_auto,q_auto/`)
     - Shows `#cover-preview`, shows `#clear-cover-btn`
     - Closes the picker overlay
   - On `#clear-cover-btn` click: hide `#cover-preview`, hide `#clear-cover-btn`, clear the src
   - Style the overlay with: position fixed, inset 0, background rgba(0,0,0,0.8), z-index 1000, display grid of thumbnails, close button
   - Add these styles inline or add minimal CSS classes to admin.css
