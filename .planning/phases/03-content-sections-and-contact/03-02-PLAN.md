   - `const FORMSPREE_ENDPOINT = 'https://formspree.io/f/YOUR_FORM_ID';` (placeholder, user swaps later)
   - Export `initContact()` function that:
     a. Gets `#contact-form` element, returns early if not found
     b. Adds submit event listener with `e.preventDefault()`
     c. Calls `validateForm(form)` -- if invalid, return
     d. Disables submit button, changes text to "Sending..."
     e. Uses `fetch()` with `method: 'POST'`, `body: new FormData(form)`, `headers: { 'Accept': 'application/json' }`
     f. On success (response.ok): call `form.reset()`, clear all error states, show success toast "Message sent! I'll get back to you soon.", re-enable button
     g. On non-ok response: parse JSON errors, show error toast with message or fallback "Something went wrong. Please try again.", re-enable button
     h. On catch (network error): show error toast "Network error. Please check your connection and try again.", re-enable button
   - `validateForm(form)` function:
     a. Select all `.contact__input[required]` fields in the form
     b. For each field, check `field.validity.valid`
     c. If invalid: add `contact__input--error` class, set sibling `.contact__error` textContent to appropriate message (use field.validationMessage or custom messages: "Name is required", "Please enter a valid email", "Please select an event type", "Please select a date", "Please enter a message")
     d. If valid: remove error class, clear error text
     e. Return boolean
   - `showToast(type, message)` function:
     a. Remove any existing `.toast` element
     b. Create div with class `toast toast--{type}`, set `role="status"` and `aria-live="polite"`
     c. Set textContent to message
     d. Append to document.body
     e. `requestAnimationFrame(() => toast.classList.add('toast--visible'))` for slide-in
     f. `setTimeout` at 5000ms: remove `toast--visible`, on `transitionend` remove element
